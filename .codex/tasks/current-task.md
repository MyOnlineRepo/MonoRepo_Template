# Current Task

## Ziel

Diese Datei dient als leichtgewichtiger Arbeitskontext für die aktuell bearbeitete Aufgabe.

## Verwendung

Sie soll nur dann gepflegt werden, wenn sie für die laufende Arbeit tatsächlich hilft. Sie ist kein Pflichtprotokoll für jede kleine Änderung.

## Inhalt

Wenn sie genutzt wird, sollte sie knapp bleiben und nur die wichtigsten Punkte enthalten:

- Ziel der aktuellen Aufgabe
- betroffener Bereich oder betroffene Dateien
- wichtige Annahmen
- offene Fragen
- geplante oder ausgeführte Checks

## Leitlinie

- kurz halten
- nur aktuelle Informationen eintragen
- keine ausufernde Verlaufsdokumentation führen
- nach Abschluss bei Bedarf wieder leeren oder überschreiben
